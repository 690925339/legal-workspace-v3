# AI法律助手 - Legal Workspace v3

一个现代化、智能化的法律案件管理系统，基于 Vue 3 (CDN) 构建，提供全方位的案件管理和AI辅助功能。

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Vue 3](https://img.shields.io/badge/Vue-3.x-brightgreen.svg)](https://vuejs.org/)
[![D3.js](https://img.shields.io/badge/D3.js-7.x-orange.svg)](https://d3js.org/)

## ✨ 核心特性

### 📋 案件管理
- **完整的案件生命周期管理** - 创建、编辑、查看、删除案件
- **多维度案件信息** - 基础信息、案情描述、当事人、证据、财务等
- **分区编辑模态框** - 针对不同信息模块的独立编辑界面
- **多当事人支持** - 支持添加多个原告、被告、第三人
- **联络人管理** - 独立的联络人信息管理
- **智能搜索与筛选** - 快速定位目标案件

### 🤖 AI智能功能
- **AI智能分析**
  - 胜诉率预测（可视化展示）
  - 风险点智能识别
  - 诉讼策略建议
- **AI对话助手**
  - 基于案件材料的智能问答
  - 快捷建议功能
  - 用户/AI头像区分
  - 增强的聊天界面
- **AI证据分析**
  - 证据完整度评估
  - 证据优先级排序
  - 智能证据收集建议
  - 证据清单导出

### 📊 数据可视化
- **关系洞察** - D3.js驱动的关系图谱可视化
  - 人物/公司关系网络
  - 资金往来可视化
  - 交互式节点拖拽和缩放
- **案件时间轴** - 关键事件时间线展示
- **财务数据展示** - 诉讼标的、律师费、诉讼费等

### 🔍 法律研究
- **智能搜索** - 法律法规和案例检索
- **高级筛选** - 多维度筛选条件
  - 法院层级、地区、裁判年份
  - 审理程序、文书类型
- **结果分类** - 法规和案例分类展示

### 👤 用户管理
- **个人资料管理** - 完整的个人信息编辑
- **头像上传** - 支持自定义头像
- **账号安全** - 密码修改功能
- **偏好设置** - 通知和系统设置

## 📖 使用说明

### 案件管理

#### 创建案件
1. 点击案件列表右上角"新建案件"按钮
2. 填写5个部分的信息：
   - **基础信息**: 案件名称、案由、法院等
   - **当事人信息**: 支持多个原告/被告
   - **案情描述**: 案情摘要、争议焦点、客户诉求
   - **财务信息**: 标的额、律师费等
   - **联络人信息**: 联系人详细信息
3. 点击"保存"或"保存并新建"

#### 查看案件详情
- 点击案件列表中的案件行进入详情页
- 左侧标签页导航，右侧内容展示
- 支持的标签页：
  - 基础信息
  - 案情描述
  - 当事人信息
  - 证据管理（证据清单、AI证据分析）
  - 财务信息
  - 高级功能（AI分析、AI助手、关系洞察、时间轴）

#### 编辑案件信息
- 在各个标签页点击右上角"笔"图标
- 打开对应的编辑模态框
- 修改后点击"保存"（带保存图标）

### AI功能使用

#### AI智能分析
1. 进入案件详情页
2. 选择"高级功能" > "AI分析"
3. 查看：
   - 胜诉率预测（大字体显示）
   - 风险点提示（颜色编码）
   - AI策略建议
4. 点击右上角"刷新分析"重新生成

#### AI对话助手
1. 进入案件详情页
2. 选择"高级功能" > "AI对话助手"
3. 使用方式：
   - 点击快捷建议快速提问
   - 在输入框输入问题
   - 按Enter发送，Shift+Enter换行

#### AI证据分析
1. 进入案件详情页
2. 选择"证据管理" > "证据分析规划"
3. 查看：
   - 证据完整度评估
   - 证据清单（按优先级排序）
   - 收集状态标记
4. 点击"导出清单"导出Excel/PDF

### 关系洞察
1. 进入案件详情页
2. 选择"高级功能" > "关系洞察"
3. 交互操作：
   - 拖拽节点改变位置
   - 滚轮缩放图谱
   - 点击节点查看详情
4. 点击"刷新图谱"重新布局

### 法律研究
1. 点击侧边栏"法律研究"
2. 输入关键词搜索
3. 点击"筛选条件"展开高级筛选
4. 选择筛选条件后点击"应用筛选"
5. 在"法规"和"案例"标签页查看结果
